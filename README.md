# Exercício de Programação em Java

Este repositório contém um exercício simples desenvolvido durante meus estudos em Ciência da Computação.

## Objetivo
Praticar conceitos básicos de programação utilizando a linguagem Java, incluindo:
- Entrada de dados pelo usuário
- Uso da classe Scanner
- Operações matemáticas
- Exibição de resultados no terminal

## Descrição
O programa solicita ao usuário dois números inteiros pelo terminal, realiza a multiplicação entre eles e exibe o resultado na tela.

## Tecnologias utilizadas
- Java
- Classe Scanner para entrada de dados
